Metodología

  • 01. Toma de contacto

    Cada negocio es un mundo, por ese motivo nos gusta conocer a nuestros clientes y las particularidades de su negocio para tomar el camino correcto y llegar al resultado que mejor se adapte a sus necesidades.

  • 02. Planificación

    Tras definir los objetivos del proyecto, diseñadores y equipo de desarrollo se reúnen y se establecen unas fechas límite para cada una de las fases en función de los requisitos del cliente.

  • 03. Desarrollo

    A medida que el proyecto avanza se envían muestras al cliente y se le otorga acceso a un servidor de pruebas para que esté al corriente de todo.

Herding Cats, Web Corporativa

Herding Cats es un estudio de desarrollo de videojuegos situado en Barcelona. Todos los elementos de diseño en este proyecto vienen de la mano de Sergio Castaño, cliente y director de arte en este caso.

La web es completamente adaptativa y fue supervisada por el cliente para asegurarse que cumplía con las expectativas. Gracias a la comunicación directa con el cliente el proyecto se completó en tan sólo unos días.

Súper Securitas

Súper Securitas es un proyecto ideado y realizado para Securitas Direct en el que los equipos comerciales se apuntan a una liga de entregados protectores de casas y barrios de todo el país.

El diseño obra del estudio Olloqui Studio y las ilustraciones de Xavi Ramiro. Fast&Yours se ocupó en este caso de la programación front-end. El requisito principal del cliente es que la web fuera adaptativa a nivel de escritorio y tablet.

Finques Urgell, portal inmobiliario

Proyecto para la inmobiliaria Finques Urgell de Barcelona.

El portal dispone de diseño responsive que se adapta de forma fluida a nivel de escritorio, tablet y móvil para una experiencia de usuario cómoda en cualquier plataforma. La página web funciona sobre la plataforma WordPress y permite a los agentes gestionar fácilmente los inmuebles en la base de datos.
